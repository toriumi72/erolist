# stores/ README

このディレクトリは **アプリ全体で共有する状態管理**を置きます。

## 方針（重要）

- グローバル状態ライブラリは使わない（必要時は **React Context + useReducer** 等で最小構成）
- 「本当にグローバルにする必要がある状態」だけを置く

## ここに置くもの

- 認証状態（将来導入時）
- UI の全体設定（テーマ、表示モードなど）
- 複数ページ横断で必要な一時状態

## ここに置かないもの

- サーバー由来データのキャッシュ（→ `useSWR` などで取得側に寄せる）
- ページ固有の状態（→ そのページ内で管理）
